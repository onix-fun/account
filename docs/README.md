# Документация сервиса Account

Добро пожаловать в документацию сервиса `account`. Этот сервис обеспечивает управление личностью пользователей, аутентификацию и профили в экосистеме Unlim.

## Содержание

- [**Архитектура**](ARCHITECTURE.md): Проектирование системы и управление процессами.
- [**Внутреннее устройство**](INTERNAL.md): Детали реализации бэкенда и фронтенда.
- [**API Контракты**](API.md): Контракты OpenAPI и gRPC сервисов.
- [**Конфигурация**](CONFIGURATION.md): Переменные окружения и настройки рантайма.
- [**Запуск и развертывание**](STARTUP.md): Инструкции по сборке и запуску сервиса.

## Ключевые возможности

- **Аутентификация**: Многоэтапный вход, регистрация, сброс пароля и управление сессиями.
- **Профили**: Управление аватарами (S3), обновление имен и почты.
- **Безопасность**: Auth на базе JWT, mTLS для gRPC, защита от CSRF и ограничение частоты запросов (rate limiting).
- **Доставка**: Единый Docker-образ с Nginx в качестве шлюза и управлением через s6-overlay.
